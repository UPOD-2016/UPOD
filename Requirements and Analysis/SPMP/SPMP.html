    4.2.3 Software Test Plan

    • The application will be tested through unit tests and by the SQA team. The unit tests will validate the operation of the models (entity objects - responsible for data persistence and retrieval). The SQA team will perform integration tests - validating the front-end interface works as expected.

    4.3 User Documentation

    The documentation will be developed on an on-going basis. Developers who write the code will be responsible for documenting the purpose of the code. Additionally, documentation/tutorials will be produced to demonstrate how to operate the website.
